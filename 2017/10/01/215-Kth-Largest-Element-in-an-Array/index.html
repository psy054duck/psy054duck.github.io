215. Kth Largest Element in an Array

Description

Find the kth largest element in an unsorted array. Note that it is the kth largest element in the sorted order, not the kth distinct element.

For example,
Given [3,2,1,5,6,4] and k = 2, return 5.

Note:
You may assume k is always valid, 1 ≤ k ≤ array’s length.

Solution 1

题目很简单，就是在数组当中找到第k大的数。

最简单的方法就是调用库函数执行$O(nlogn)$的排序然后用索引得到第k大的数。

Solution 2

另外还有一个简单粗暴的方法，用堆排序。之所以简单粗暴，是因为我们依然可以调用库函数。

Solution 3

第三种解法是利用快排的思想，与快排不同的是每次分割之后，我们只处理其中一半的数据，而不是像快排一样要处理两段数据。因此在速度上可以快过单纯的快排。当然与快排一样，我们可以通过一些手段在系数上提高算法效率，例如在切割至数据量不大的时候使用选择排序来找到目标。选择pivot的时候不是单纯的使用同一位置的数，而是选取三个数并且使用三个数中的中位数来充当pivot。

该算法的关键在于每次为了找到第K大的数，在切割出的子问题中也许就不是第K大。例如在一个10个数字的数组中寻找第7大的数，第一次选取的pivot在经过计算后发现它是第5大的数，此时需要在[6:10]中继续寻找，但此时应该在这个子数组中寻找第2大的数而不是第7大。
