64. Minimum Path Sum

Description

Given a m x n grid filled with non-negative numbers, find a path from top left to bottom right which minimizesthe sum of all numbers along its path.

Note: You can only move either down or right at any point in time.

Given the above grid map, return

Solution

这是一道动态规划类型的题目。这道题的状态转移方程很好确认，因为按照题目的意思只要移动一步就进入了下一个状态。

​ $$mmap[i][j] = min(mmap[i+1][j], mmap[i][j+1]) ＋ grid[i][j]$$

其中mmap[i][j]表示从第(i, j)格子移动到右下角格子所需要开销。所以这题只需要计算出mmap[0][0]即可。上述方程很明白，要计算mmap[i][j]只需要计算两个移动之后的开销加上grid[i]j的开销即可。
