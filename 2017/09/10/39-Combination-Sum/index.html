39. Combination Sum

Description

Given a set of candidate numbers (C) (without duplicates) and a target number (T), find all unique combinations in C where the candidate numbers sums to T.

The same repeated number may be chosen from C unlimited number of times.

Note:

  • All numbers (including target) will be positive integers.
  • The solution set must not contain duplicate combinations.

Solution

从题目描述来看很显然这题可以用回溯来解决。如果把对解的查找表示为树的话，回溯法就相当于对这棵树进行深度优先搜索同时配合上剪枝，明显提高算法效率。当然，这道题本身也不存在比回溯更加暴力的算法了，不像八皇后对元素的个数做了限制，所以八皇后最暴力的解法是每次都遍历8个数字后才判断是否符合要求，即便如此算法总能停止。而本题对数字个数没有限制，所以查找停止的条件就只剩下判断当前查找的数字和是否大于target，如果连这个都不做判断那么程序就永远不会停止了。

由于需要得到具体的解集，因此需要一个vector<int> buffer用于临时表示解集中的每个解，递归栈中的每一层函数都要尝试往这个buffer中放入一个数字，并且跳过不符合要求的数字。也可以认为每一层递归当时在解决上一层函数的子问题。以题目中的数据为例，当buffer中的数据为[2]时，那么当前函数就要从[2, 3, 6, 7]中找到和为7-2＝5的数字，将这些数字与buffer中的数字结合就是所要的解。
